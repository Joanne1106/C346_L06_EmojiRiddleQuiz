# Emoji Riddle Quiz
Emoji Riddle Quiz – Guess movies, foods, drinks, and landmarks from images! Select your answer, submit to see if you’re correct or wrong, and restart the game anytime.

# Features
Image-based questions – Guess movies, foods, drinks, and landmarks from pictures.

Multiple choice answers – Pick the correct option from a list.

Immediate feedback – Shows if your answer is correct (green border) or wrong (red border) after submission.

Score tracking – Displays your total score at the end of the quiz.

Restart quiz – Shuffle and start with new random questions anytime.

Randomized questions – Each game presents a different set of questions from the pool.
